## Features

- 📂 Scan any folder
- 🧠 Smart labeling (Invoices, Contracts, Projects, etc.)
- ✨ Suggest cleaner filenames
- 🔍 Detect duplicate files using hashing
- 👀 Preview changes before applying
- 📊 Export analysis report (TXT / JSON)

## Tech Stack

- Python
- Tkinter (GUI)
- File system + hashing

## How It Works

The app analyzes file names and extensions, applies smart rules to classify them, and suggests a clean folder structure.
